## 电费监控系统 v2 — 真·实时数据抓取

用户提供了辰域智控付款二维码截图，经扫码解码后获知 URL 为：
`http://www.wap.cnyiot.com/nat/nat.aspx?id=P+KgcBqh/9/gOpbR+039FA==`

发现加上 `&by=a` 参数可绕过微信认证直接获取 HTML 页面，从中正则提取实时数据。

**当前数据（2026-07-20 20:45）：**
- 余额：¥31.86
- 电量：32.61 kWh
- 电价：0.977 元/kWh

**系统架构：**
1. `server.js` — Node.js 后端
   - 每请求一次 `/api/meter` → 从 cnyiot.com 抓取最新数据
   - 自动跟随 302 重定向（nat.aspx → pay.aspx）
   - 正则解析 HTML 提取：表名、表号、剩余电量、余额、电价
   - 数据保存到 `data.json`（JSON 文件存储，只在新数据变化时追加）
   - 提供 `/api/meter`、`/api/history`、`/api/history/clear` 三个端点
   
2. `index.html` — 前端面板
   - 自动调用 `/api/meter` 加载实时数据
   - KPI 卡片：余额、电价、预估可用天数
   - 快捷统计：今日/昨日/日均/日均花费
   - 两个趋势图（Chart.js）：余额变化 + 电量消耗
   - 历史记录表格（时间、余额、电量、日消耗、日花费）
   - 支持"手动记录"补充数据
   - 深色/浅色主题切换
   - 低余额预警（可设阈值）

**技术亮点：**
- 零认证！无需登录/密码/Token
- Node.js 原生 http 模块（零依赖），一个文件搞定
- 数据仅存本地 JSON，不经过第三方服务器
- 自动去重：同一数值不重复记录
- 重定向跟随 + 多级容错

**文件：**
- `server.js` — Node.js 后端
- `index.html` — 前端面板
- `data.json` — 历史数据存储
- 运行：`node server.js` → http://localhost:8080/

**发现的备用数据源：**

- ShowDoc: zyiot API — `/apiOther/powerData` 批量获取电量数据
